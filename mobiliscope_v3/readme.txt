Le dossier "mobiliscope_v3" contient l'ensemble des fichiers nécessaires pour la préparation des données et pour la géovisualisation. 

Le Mobiliscope peut être reproduit pour Montpellier en utilisant le jeu de données brutes 
mis à disposition par Montpellier Méditerrannée Métropole sous license ODbL et les scripts R de l'équipe du Mobiliscope.


Contenu des sous-dossiers :
- "opendata" : données sources issues de l'EDGT Montpellier 2014

- "scriptsr" : 
	- scripts R utilisés pour la préparation des données du Mobiliscope
	- "data" : bases de données utilisées ou produites sous R
	- "shp" : couches d'information géographique utilisées ou produites sous R
	- "txt" : fichiers csv utilisés ou produits sous R

- "www" : 
	- ensemble de fichiers déposés sur le serveur web du Mobiliscope à l'exception du dossier "data" qui ne contient que les données de Montpellier
